Mijn Projecten

Project Text Adventure - C# Console Application - 2023

Bij Project Text Adventure was de doel om een spel te maken met tekst in de console van Visual Studio. Met dit project heb ik best wel veel geleerd en ik had er ook veel aandacht in gestopt, de uitkomst was ik ook echt blij mee. Ik had veel gebruik gemaakt van Ascii art, ik heb bijna alle Ascii art zelf gevonden en veranderd als het nodig was, ik heb Ascii art gebruikt dat op een openbare website stond en het game is voor nu niet toegankelijk.

Project Arcade Game - C# Unity - 2024

Bij Project Arcade Game was de doel om een arcade type spel te maken voor een doelgroep met een limitatie, mijn groep kreeg de doelgroep mensen met een beperking en als limitatie dat de spel een resolutie moest hebben van 128x128. Een heftige overeenkomst maar ik was nogsteeds wel blij met de uitkomst.

Project Micro Game - C# Unity - 2024

Bij micro game was de doel om een kleine dus micro spel te maken die je kunt spelen op de telefoon, wij hadden sinds de start al gekozen voor een Aap/Banaan thema zoals je ziet op de logo. We hebben de game niet echt op het eind kunnen optimaliseren en het was soms traag, maar we waren echt blij met het resultaat. Klik op de knop om er meer over uit te vinden.
